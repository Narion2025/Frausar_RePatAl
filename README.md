# Frausar_RePatAl: Analyse- und Visualisierungstool für Beziehungschroniken

## Vision und Zielsetzung

Dieses Projekt entwickelt ein interaktives Analyse-Dashboard zur quantitativen und qualitativen Auswertung von Beziehungsprozessen auf Basis von Kommunikationsdaten (z.B. Chats, Transkripte). Das Tool visualisiert die Entwicklungen über die Zeit und stellt Marker für Beziehungsgesundheit, Manipulation, Ressourcenwachstum und Krisenentwicklung dar.

Das Ziel ist ein Werkzeug, das sowohl für Experten als auch für reflektierte Paare zur Selbstanalyse geeignet ist.

## Setup-Anleitung (Initial)

1.  **Repository klonen:**
    ```bash
    git clone https://github.com/Narion2025/Frausar_RePatAl.git
    cd Frausar_RePatAl
    ```

2.  **Virtuelle Umgebung erstellen und aktivieren:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

3.  **Abhängigkeiten installieren:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Anwendung starten (zukünftig):**
    ```bash
    python frontend/app.py
    ```
